Description

Le Concept

ImmoSync Web est l'adaptation de la solution de gestion immobilière en version web. Elle cible les techniciens sur le terrain et les gestionnaires qui ont besoin d'un accès depuis n'importe quel appareil.

L'objectif est de proposer une interface responsive et intuitive, avec un tableau de bord en temps réel permettant de suivre les chantiers, les biens et les interventions depuis un navigateur, sans installation.

Capacités

Fonctionnalités Clés

Interface Responsive

Interface adaptée à tous les appareils — ordinateur, tablette, smartphone — pour une utilisation terrain optimale.

Tableau de Bord

Vue synthétique des chantiers en cours, des interventions planifiées et des statuts en temps réel.

Gestion Multi-Rôles

Interface adaptée selon le profil connecté : Administrateur, Gestionnaire, Technicien ou Propriétaire.

Persistance PDO

Accès sécurisé à la base de données MySQL via PDO, avec requêtes préparées pour prévenir les injections SQL.

Technique

Architecture & Stack

PHP & Bootstrap

Structure MVC en PHP natif, avec un front-end Bootstrap pour une interface responsive et cohérente. La couche d'accès aux données utilise PDO pour sécuriser les échanges avec MySQL.
