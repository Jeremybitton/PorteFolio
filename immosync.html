Description

Le Concept

ImmoSync (GestTravaux) est une application desktop développée pour simplifier la gestion des biens immobiliers et le suivi rigoureux des chantiers de rénovation.

Le projet répond à un besoin de centralisation des informations entre les propriétaires, les inspecteurs et les entrepreneurs. Il permet de suivre l'évolution d'un projet de sa création (devis, planification) jusqu'à sa livraison finale.

Capacités

Fonctionnalités Clés

Gestion Multi-Rôles

Interface adaptée selon le profil : Administrateur, Gestionnaire, Inspecteur, Entrepreneur ou Propriétaire.

Suivi de Chantier

Visualisation de l'état d'avancement des travaux, gestion des prestations et des plannings.

Espace Sécurisé

Système d'authentification robuste avec gestion fine des habilitations (RBAC).

Persistance des Données

Base de données MySQL centralisant les biens, les utilisateurs et l'historique des travaux.

Architecture & Stack
