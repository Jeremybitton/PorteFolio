Contexte

Le Projet

Développement d'un outil professionnel d'extraction et de structuration des règles de sécurité CIS (Center for Internet Security) à partir de fichiers PDF.

L'objectif est d'automatiser la comparaison entre différentes versions de benchmarks (ex: Oracle Linux 9 vs 10) et de générer des rapports d'audit exploitables pour les équipes sécurité.

Fonctionnalités Clés

Extraction PDF

Analyse intelligente de fichiers PDF complexes pour extraire les règles de sécurité, leur sévérité, et les descriptions techniques.

Comparateur

Algorithme de comparaison ("Diff") pour identifier les ajouts, suppressions et modifications entre deux versions de benchmarks.

Rapports

Génération automatique de fichiers JSON structurés et de tableaux Excel formatés pour les auditeurs.

Mise à jour

Module de mise à jour dynamique pour appliquer les changements et remplacer les variables (ex: années) dans les documents finaux.

Interface Utilisateur

Dashboard Streamlit

Une interface web interactive développée avec Streamlit pour faciliter l'usage de l'outil par les équipes non-techniques.
